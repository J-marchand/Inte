Au cours de cette formation nous allons voir 2 matières :

l'intégration

Le HTML :

Le HTML sert à créer le contenu du site : elle va0 contenir contenir uniquement la partie textuelle du site.

Le CSS :

Le CSS va nous permettre d'ajouter la couche graphique à notre texte : les couleurs, les formes, le fond, etc.

Le dévelopement

Le Javascript :

Javascript est un language de programmation de script employé pour rendre une page web interactive, le comportement de page et son apparence pourra varier en fonction de certains paramètres.

Le PHP :

Le PHP est un language de programmation libe, utilisé pour générer des pages web dynamiques, qui sont donc modifiés en fonction de certaines informtations (heure, nom de l'utilisateur, formulaire rempli par l'utilisateur etc).

MySQL

MySQL est un systeme de gestion de base de données, il permet d'enregistrer des informationqs rempli par l'utilisateur (exemple :formulaire de création de compte) pour ensuite etre ré-utilisé au moment approprié (ex: connexion à un site internet avec identifiant et mot de passe).
